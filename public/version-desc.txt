# Quick Start

## Windows
1. Extract the archive
2. Run `discord-quest-helper.exe` while Discord is running in the background
3. Click `Auto Detect Token` to detect your Discord accounts
4. Filter quests and click `Start` to finish the quest

## MacOS
1. Download the latest `.dmg` Filter
2. Open the `.dmg` file and drag the app to your Applications folder
3. Run `Discord Quest Helper` from Applications
